GreenOrbit Early Warning Dashboard - Project Setup Guide

FEATURES IMPLEMENTED:
=====================

✅ FARMER DASHBOARD
   - Home Summary Card with Crop Health, Pest Risk, Disease Risk
   - Early Warning Alerts with pest names, risk levels, timeframes
   - Interactive Map View with farm boundaries
   - Explainable AI Panel showing NDRE, Soil Moisture, Temperature, Humidity analysis
   - "What To Do" Panel with scouting, prevention, and control advice
   - Real-Time Sensor Status with live IoT data (soil moisture, temperature, humidity)
   - Farm selector to switch between multiple farms
   - WebSocket connection for live sensor updates

✅ ADMIN DASHBOARD
   - System Overview with key metrics
   - Feature Importance visualization for ML model
   - Alert Management with active/resolved filtering
   - Model Management with risk thresholds
   - Farmer Feedback section placeholder
   - Real-time data monitoring capabilities

✅ AUTHENTICATION & ROUTING
   - Role-based access control (Farmer vs Admin)
   - Login page with role selection
   - Protected routes
   - Session persistence with localStorage

✅ BACKEND API
   - Express.js REST API server
   - WebSocket server for real-time sensor streaming
   - Mock data endpoints for all resources
   - Sensor data simulation every 10 seconds
   - Farm subscriptions for targeted updates

✅ INDICATOR SYSTEM (FROM EXCEL)
   - NDRE classifications (Very Healthy to Severe Stress)
   - Soil Moisture classifications (Low, Moderate, High)
   - Temperature classifications (Low to Very High)
   - Humidity classifications (Low, Medium, High)
   - Risk assessment based on multiple indicators
   - Pest-specific risk models for 7 major maize pests

PEST MODELS INCLUDED:
   1. Fall Armyworm (FAW) - 7-10 day warning
   2. Aphids - 5-7 day warning
   3. Stem Borers - 10-14 day warning
   4. Thrips - 5-8 day warning
   5. Termites - 14-21 day warning
   6. Cutworms - 7-10 day warning
   7. African Armyworm - Variable warning

INSTALLATION:
==============

1. Navigate to the project directory:
   cd greenorbit-dashboard

2. Install dependencies:
   npm install

3. Install backend dependencies:
   npm install express cors ws

4. Create .env file (copy from .env.example):
   cp .env.example .env

RUNNING THE APPLICATION:
=========================

Development Mode:
   npm run dev

Or run separately:
   # Terminal 1 - Frontend
   npm start

   # Terminal 2 - Backend
   npm run backend

Production Build:
   npm run build

API ENDPOINTS:
==============

Authentication:
   POST   /api/auth/login
   POST   /api/auth/logout

Farms:
   GET    /api/farms
   GET    /api/farms/:farmId

Alerts:
   GET    /api/alerts
   GET    /api/alerts/:farmId

Historical Data:
   GET    /api/historical-data/:farmId?days=30

Admin:
   GET    /api/admin/stats
   GET    /api/admin/feature-importance

WebSocket:
   ws://localhost:5000
   - Messages: { type: 'subscribe'|'unsubscribe', farmId: 'FARM-XXX' }
   - Updates: { type: 'sensor-update', farmId, data, timestamp }
